Installation
============

Prepare your environment, then download and build AthenaK.

Step 1: confirm system tools
----------------------------

.. code-block:: bash

   cmake --version
   c++ --version

If you plan to build with MPI:

.. code-block:: bash

   mpicxx --version

Step 2: review dependencies
---------------------------

- :doc:`requirements`
- :doc:`download`

Step 3: configure and compile
-----------------------------

Continue with :doc:`build` to configure and compile AthenaK.
